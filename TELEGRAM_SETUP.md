# Настройка Telegram Mini App

## Шаг 1: Развертывание приложения

1. Разверните собранное приложение на вашем домене
2. Убедитесь, что приложение доступно по HTTPS (обязательно!)
3. Telegram требует HTTPS для Mini Apps

## Шаг 2: Настройка бота в BotFather

1. Откройте Telegram и найдите бота [@BotFather](https://t.me/BotFather)

2. Отправьте команду `/mybots` и выберите вашего бота

3. Выберите **"Bot Settings"** → **"Menu Button"** → **"Configure Menu Button"**

4. Отправьте BotFather URL вашего развернутого приложения:
   ```
   https://ваш-домен.com
   ```

5. После этого отправьте название кнопки, например:
   ```
   Открыть приложение
   ```

## Шаг 3: Настройка Web App

1. В чате с @BotFather выберите ваш бот
2. Выберите **"Bot Settings"** → **"Menu Button"** → **"Edit Menu Button URL"**
3. Введите URL вашего приложения

## Альтернативный способ - через команду

Вы также можете настроить Mini App через команду в боте:

1. Отправьте команду `/newapp` в @BotFather
2. Выберите вашего бота
3. Введите название приложения
4. Введите описание
5. Загрузите иконку (640x360 PNG)
6. Загрузите GIF-превью (необязательно)
7. Введите URL вашего приложения

## Шаг 4: Тестирование

1. Откройте вашего бота в Telegram
2. Нажмите на кнопку меню (иконка с четырьмя квадратами рядом с полем ввода)
3. Приложение должно открыться внутри Telegram

## Важные моменты

- **HTTPS обязателен**: Telegram требует, чтобы все Mini Apps были доступны только по HTTPS
- **Аутентификация**: Приложение автоматически получает данные пользователя из Telegram
- **Тестирование**: Сначала протестируйте приложение в браузере, затем в Telegram
- **Дебаг**: Для отладки в Telegram Desktop можно использовать DevTools (Ctrl+Shift+I)

## Проверка интеграции

Приложение определяет, запущено ли оно в Telegram Mini App, и автоматически:
- Получает данные пользователя (имя, фото, username)
- Адаптирует интерфейс под среду Telegram
- Показывает "Telegram Mini App" в заголовке при запуске через Telegram

## Что дальше?

После настройки вы можете:
- Отправить ссылку на бота друзьям
- Опубликовать бота в каталоге ботов
- Добавить встроенный режим (inline mode)
- Настроить уведомления

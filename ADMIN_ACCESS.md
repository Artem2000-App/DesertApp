# Доступ к панели администратора

## Как открыть панель администратора базы данных

Панель администратора доступна по адресу `/admin`:

**Локальная разработка:**
```
http://localhost:5173/admin
```

**Продакшн:**
```
https://ваш-домен.com/admin
```

## Возможности панели администратора

### Таблицы
- Просмотр всех таблиц базы данных (users, products, orders, clients, categories, order_items)
- Поиск по данным в таблицах
- Просмотр детальной информации о записях
- Удаление записей
- Обновление данных в реальном времени

### Storage
- Просмотр всех файлов в трёх бакетах:
  - **product-images** - Фото товаров (публичный доступ)
  - **order-attachments** - Документы заказов (приватный доступ)
  - **user-content** - Контент пользователей (приватный доступ)
- Загрузка новых файлов
- Удаление файлов
- Просмотр размера файлов

## Безопасность

⚠️ **Важно**: Панель администратора не имеет дополнительной аутентификации и доступна всем, кто знает URL `/admin`.

Для продакшн-среды рекомендуется:
1. Добавить дополнительную аутентификацию для доступа к панели администратора
2. Использовать отдельный защищённый домен для админки
3. Настроить ограничения доступа на уровне базы данных через RLS политики
4. Ограничить доступ к маршруту `/admin` на уровне веб-сервера

## Выход из панели администратора

Нажмите кнопку "Выйти" в правом верхнем углу панели администратора, чтобы вернуться к обычному интерфейсу приложения.
